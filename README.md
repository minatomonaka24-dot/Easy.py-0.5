# 🎨 Ultra Multi-Language Custom Programming Language

An advanced custom mini-programming language interpreter built with Python and Tkinter, designed to work smoothly on Android (Pydroid 3). It now supports over 14+ massive commands in multiple languages (Japanese, English, Vietnamese) with zero encoding bugs!

## ⚡ Massive Features & Commands (主な機能と大量のコマンド)

### 📢 Sound & Effects (音とメッセージ演出)
- `音` / `beep` / `tiếng` : Plays a system buzzer sound on your device.
- `警告` / `alert` / `báocáo` : Shows a warning pop-up dialog.
- `アニメ` / `type` / `chữchậm` : Displays text with a cool typewriter animation effect.

### 🔢 Advanced Math (高度な数字と計算)
- `偶数奇数` / `iseven` / `sốchẵn` : Validates if a number/variable is even or odd.
- `絶対値` / `abs` / `trịtuyệtđối` : Converts negative numbers into absolute values.
- `四捨五入` / `round` / `làmtròn` : Rounds a decimal to the nearest integer.

### 📝 Text Operations (高度な文字操作)
- `長さ` / `len` / `độdài` : Counts the length of a string.
- `置換` / `replace` / `thaythế` : Replaces specific words within a text.
- `大文字` / `upper` / `chữhoa` : Converts text to UPPERCASE.
- `小文字` / `lower` / `chữthường` : Converts text to lowercase.

### 🛠️ System & Utilities (システム・便利ツール)
- `入力` / `ask` / `nhập` : Prompt user to input text via dialog box.
- `合体` / `join` / `ghép` : Combines variables and text seamlessly.
- `乱数` / `rand` / `ngẫunhiên` : Generates a random number between given ranges.
- `パスワード` / `password` / `mậtkhẩu` : Automatically generates a secure random password.
- `現在時刻` / `time` / `thờigian` : Fetches current system time (HH:MM:SS).
- `アプリ終了` / `close` / `đóng` : Closes the running GUI application instantly.

---

## 🚀 How to Run (動かし方)
1. Install **Pydroid 3** on your Android device.
2. Copy the Python code (`Easy.py`) into your editor.
3. Tap the **RUN** button and enjoy!
